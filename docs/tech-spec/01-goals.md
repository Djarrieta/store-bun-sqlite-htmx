# §1 — Goals and non-goals

## Goals

1. Build a functional online store for **Cristaterapia (Cristal Therapeutics)** — a real operating business.
2. Admin panel to manage orders, products, categories, images, and a future **manual AI assistant**.
3. Chat module for **payment proof** (Nequi): the customer uploads the transfer image and the assistant links it to the order.
4. WhatsApp integration as a **future channel** (same AI backend).
5. Admin **NL → SQL** analytics over database views (no PII).
6. Public, simple, mobile-first storefront with a clean UI.

## Non-goals for v1

- Real-time payments (payment gateway); the initial payment method is **manual Nequi transfer** with staff verification.
- Multi-language (Spanish only).
- Native mobile apps.
- Marketplace model.
- Complex tax/invoicing.
